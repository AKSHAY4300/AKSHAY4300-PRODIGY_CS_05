# 🌐 Network Packet Analyzer (Python)

A Python-based network packet analyzer that captures and inspects live network traffic. This tool was developed as part of my cybersecurity internship to understand how data flows across networks and how packets can be analyzed in real time.

---

## ⚠️ Ethical Use Notice

This project is intended strictly for **educational purposes**.  
It should only be used on:
- Your own system  
- Authorized networks  
- Controlled lab environments  

Unauthorized packet sniffing may violate privacy and legal regulations.

---

## 🚀 Features

- Captures live network packets
- Displays:
  - Source IP address
  - Destination IP address
  - Protocol type (TCP / UDP / ICMP)
- Real-time packet analysis
- Lightweight CLI-based tool

---

## 🧠 Concepts Covered

- Network Packet Structure
- IP Layer Analysis
- Protocol Identification
- Packet Sniffing Fundamentals
- Ethical Considerations in Network Monitoring

---

## 🛠️ Tech Stack

- Python 3
- Scapy (Packet Manipulation Library)
- Use Kali Linux
